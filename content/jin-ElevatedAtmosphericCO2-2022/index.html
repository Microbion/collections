    jin ElevatedAtmosphericCO2 2022

    • 磷对于土壤非常重要，生物对有机磷的矿化可能是主要过程
    • 气候变化改变有机磷转化。二氧化碳浓度升高提高了土壤有机磷浓度
    • 在高CO2浓度的条件下有机磷加速矿化的微生物机制尚不清楚
    • 本研究量化了微生物对高CO2矿化有机磷的贡献。假设：
      1. eCO2增加了根际有机磷矿化，由于相关的微生物系统类型和代谢潜力改变导致
      2. 高浓度CO2加强了P匮乏土壤中微生物对P的利用，一定程度上微生物的系统发育和功能潜力发生了显著变化

    微生物对P需求的增加有利于P矿化¶

    CO2↑ => 根系分泌物↑ => 生物可利用C↑ => 微生物P利用率↑ =>

    根系分泌物↑ => 微生物生物量↑ => P需求↑ => C限制转为P限制 [[202203071542#Fig2]]

    高浓度CO2下植酸矿化加快，此外，植酸矿化程度的增加和P限制有利于根际磷代谢更有利的微生物种群

    P矿化相关的微生物分类和代谢属性，和微生物P利用特性¶

    β多样性，而不是α多样性与P矿化相关。Vertosol类型土中α多样性没有变化。在Chromosol土显著，在Vertosol土略有显著。所以，可能有共同代谢特征的细菌类群响应CO2和P限制，phylofactor分析发现Bacteriodetes和Gemmatimonadetes在eCO2条件下升高，并且与植酸矿化正相关

    在Bacteroidetes中，Chitinophagaceae，Microscillaceae，Sphingobacteriaceae and Hymenobacteraceae起作用。Chitinophagaceae发现可以矿化几丁质和纤维素，产生β-葡萄糖苷酶。Gemmatimonadaceae适应低营养环境，被可以细胞内积累磷酸盐，适用于废水处理中的磷酸盐固定化。

    Bacteriodetes和Gemmatinomnadetes的一些科有相似的响应，编码methylgallate, protocatchuate, gallate 降解途径在两种类型的土壤中与P矿化显著相关。上诉途径参与了芳香族化合物的降解，因为植酸与芳香族化合物的C骨架结构相似，C和P的矿化可能是一个共代谢过程。

    eCO2改变了微生物代谢，导致了微生物P利用效率增加。P是核苷酸组成的重要成分，ASV预测发现磷酸戊糖途径、嘧啶脱氧核糖核苷的挽救途径在eCO2下发生变化。磷酸戊糖途径产生合成代谢反映的等价物NDPH，保护了微生物受ROS的毒性。从头合成DNA是耗能过程，因此在eCO2条件下表达嘧啶脱氧核醣回收途径能力的细菌具有明显优势。

    真菌对P矿化的贡献¶

    真菌Shannon指数与P矿化相关。Chromosol中的Agaricus和 Vertosol中的Auriculariales，均属于外生菌根，eCO2条件下相对丰度升高。 - 机制尚不确定。Chromosol土中的丛植根菌，Claroideoglomus和Funneliformis，促进植物在土壤中的竞争磷的能力，或通过增加细菌中磷酸酶基因的表达刺激细胞介导的有机磷矿化。 - 有一些P矿化相关的属涉及到有机质降解，Ascobolus和Arachnomyces，前者可以从粪便中获取营养，后者可以降解木质素和纤维素。因此可以直接分解有机物而加速植酸矿化。

    结论¶

    eCO2促进了小麦根际植酸矿化。 C efflux↑ => 刺激微生物活动，降解芳香族化物的细菌可能有助于矿化磷，此外菌根利用P也促进了细菌的矿化作用。[[202203071542#Fig9]]
